# Web-Tech-Project-Online_resume_builder
Resume Builder is a web-based application developed using PHP, MySQL, HTML, CSS, and JavaScript that enables users to create, manage, and download professional resumes. It provides secure user authentication, profile management, customizable resume sections, and database-driven storage for an efficient and user-friendly resume creation experience.

## Features

* User registration and login
* Secure authentication system
* Create and edit resume details
* Manage personal, educational, and professional information
* Generate professional resumes
* Database-driven data storage
* Responsive and user-friendly interface

## Technologies Used

* PHP
* MySQL
* HTML5
* CSS3
* JavaScript
* Bootstrap

## Installation

1. Install XAMPP.
2. Copy the project folder to `htdocs`.
3. Start Apache and MySQL from XAMPP.
4. Create a database in phpMyAdmin.
5. Import the `schema.sql` file.
6. Configure database settings in `includes/db.php`.
7. Open the project in your browser.

## Usage

1. Register a new account.
2. Log in to the system.
3. Enter your resume information.
4. Save and manage your details.
5. Generate and download your resume.
